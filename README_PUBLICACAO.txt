SGCL Landing Page - Pacote para publicação

Estrutura:
- index.html: página principal institucional/comercial do SGCL
- acesso-aplicativos.html: página de acesso ao SGCL nas nuvens
- obrigado.html: página simples de agradecimento, caso a Locaweb conecte um formulário com redirecionamento
- css/styles.css: estilos da página
- js/main.js: menu mobile, animações, validação simples do formulário e fallback via e-mail
- assets/: imagens usadas na página

Links de acesso configurados:
- Link principal: https://sgcl.masterdaweb.net/
- Link alternativo: http://sgcl.masterdaweb.net/

Formulário de demonstração:
O formulário está preparado visualmente e com validação básica em JavaScript. Hoje ele usa fallback mailto para contato@sgcl.com.br.
Para produção, a Locaweb pode conectar o formulário ao serviço de envio deles alterando o action/method do form no index.html e removendo/adaptando o listener de submit em js/main.js.

Publicação:
Enviar todos os arquivos e pastas para a raiz do site. O arquivo index.html deve ficar na raiz pública do domínio.
